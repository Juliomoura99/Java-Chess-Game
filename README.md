# ♟️ Java Chess Game

This project is a fully functional **Chess Game developed in Java**, created as the second project for the **Programming Paradigms B** course.

The main goal of this project was to develop a complete chess game using **Java** and a **Graphical User Interface (GUI)**, applying object-oriented programming concepts, game logic, event handling, and interface design.

---

## 📌 About the Project

This project was developed as an academic assignment for the Programming Paradigms B course.

The objective was to create a chess game that allows players to interact with a visual board, move pieces according to chess rules, and play a complete match using a graphical interface.

The project was built using **Apache NetBeans IDE 12.4** and follows a Java-based structure with source files organized inside the `src` folder.

---

## 🎮 Main Features

The project includes the following features:

- Graphical chess board
- Interactive chess pieces
- Two-player gameplay
- Piece movement through the interface
- Basic chess movement rules
- Automatic screen flow during program execution
- Java Swing-based interface
- Executable `.jar` file available for quick testing
